# css-handson-2025

このリポジトリは、HTMLとCSSによる視覚表現を学ぶためのデモサイトを提供します。

## 公開サイト

このサイトは GitHub Pages で公開されています:
https://seekseep.github.io/css-handson-2025/

## デプロイについて

- `main` ブランチへの push により、自動的に GitHub Pages へデプロイされます
- デプロイは GitHub Actions により自動実行されます (`.github/workflows/pages.yml` を参照)
- `public/` ディレクトリの内容が公開されます
